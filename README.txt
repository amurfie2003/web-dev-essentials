This repo contains notes on the essentials needed to produce modern web based projects from scratch. The information contained in the file is intended as a companion/reference guide while writing web based software and not explicit study material. For example while using arrays in your code you can quickly remind yourself of the key methods and properties available to you.This however will not be helpful if you do not have prior exposure to the topic. It was decided to keep the notes in a single file for ease of use when using a modern code editor.

You will find information on core concepts in: 
JavaScript 
HTML
CSS
Linux OS (command line)
Git/GitHub
Node.js
Browsers
As well as recomemded Books and resourses
